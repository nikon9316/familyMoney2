Политика конфиденциальности Family Finance

Версия: Level 5.2.1. Дата: 2026-05-06.

1. Какие данные обрабатываются

Приложение хранит Telegram ID, имя пользователя Telegram, семейную группу, кошельки, категории, операции доходов/расходов, переводы, долги, цели, бюджеты, историю изменений и технические логи.

2. Для чего используются данные

Данные используются только для ведения семейного финансового учета, формирования отчетов Excel/PDF, уведомлений, резервного копирования и администрирования сервиса.

3. Где хранятся данные

В production-режиме данные хранятся в PostgreSQL, например Railway PostgreSQL. Резервные копии сохраняются в папку BACKUP_DIR или в подключенный persistent volume.

4. Доступ к данным

Обычный пользователь видит только данные своей семьи. Админ-панель защищена session cookie, CSRF-токеном, rate limit и optional IP allowlist.

5. Передача третьим лицам

Данные не продаются и не передаются третьим лицам. При включенном Sentry отправляются технические ошибки без финансовых данных в явном виде, насколько это возможно.

6. Удаление данных

Для удаления данных администратор сервиса должен удалить семью/пользователя из базы или выполнить миграционный/административный сценарий удаления.

7. Контакты

Укажите здесь контакты владельца сервиса перед публикацией: email, Telegram или юридическое лицо.
